# PowerBI-Analysis
This dataset contains sales transaction data used for analysis and visualization in Power BI. It captures sales performance across different countries, products, and salespersons over time.
# Columns Explanation:
Sales Person – Name of the salesperson responsible for the sale
Country – Country where the sale occurred
Product – Type of product sold
Date – Date of the sales transaction
Amount – Total sales amount generated
Boxes Shipped – Number of product boxes shipped
# Purpose of the Dataset
The dataset is used to:
Analyze sales performance by country and product, Identify top-performing salespersons, Track sales trends over time, Compare sales amount vs quantity shipped, Build interactive Power BI dashboards
# Tools Used: 
Microsoft Excel – Data source, Power BI – Data cleaning, modeling, and visualization
# Use Case
This data helps business stakeholders:
Monitor regional sales growth, Optimize product distribution, Improve sales strategy and performance tracking
